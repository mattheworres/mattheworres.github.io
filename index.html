My Work

Hoot Draft

Hoot Draft (formerly PHP Draft) is an open source app written with SAAS deployment in mind that helps Fantasy Sports leagues run "offline" drafts (off-platform from the service they use in-season) in a "presenter" style mode.

Sign up for a commissioner account at hootdraft.com or check out the source code!

CompareHare

CompareHare is a (unreleased) price comparison and alert app that periodically gathers the latest pricing data on products or services the user indicated preferences for, and automatically notifies them of what they consider a good deal.

Originally written for electric and gas utility prices, it was expanded to also include retail items like large home appliances in 2022.

Source is private, but I would be happy to discuss a variety of details about this project with you!

Home Lab

Not just one project, but several! My home lab enables me to quickly and easily spin up new instances of any tech stack I'd like to tinker with or use for projects.

The current server runs on Unraid, and deploys a variety of Docker containers, including Jenkins for CI, Nginx for web serving, and any other containers I choose to spin up to run software (MariaDB, .NET hosting, and more!).

The entire instance is private and secure, but I will gladly discuss any element of this project and how it pertains to me as a software developer!
